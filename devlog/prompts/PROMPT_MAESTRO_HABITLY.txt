12. CLASE VIVIENDA — CONTEXTO DETALLADO

La clase Vivienda representa un inmueble genérico dentro del sistema.

Atributos actuales:
private static final double IGIC = 7.0;
private String direccion;
private double precioBase;
private double mensualidadTotal;
private double totalPagadoMes;
Responsabilidades actuales:
almacenar ubicación
almacenar precio base
calcular precio con impuestos
gestionar pagos
calcular deuda pendiente
comprobar si está pagado
proyectar ingresos
aplicar subidas anuales
Métodos actuales:
getDireccion()
getPrecioBase()
setDireccion(String nuevaDireccion)
setPrecioBase(double nuevoPrecioBase)
getPrecioConImpuestos()
registrarPago(double cuota)
getPendienteDePago()
isPagadoCompleto()
getProyeccionIngresos(int meses)
aplicarSubidaAnual(double porcentaje)
Lógica importante:
mensualidadTotal se inicializa con getPrecioConImpuestos()
totalPagadoMes empieza en 0
al cambiar el precioBase, debe recalcularse la mensualidad total
registrarPago() acumula pagos
isPagadoCompleto() permite pagos exactos o superiores
getProyeccionIngresos() admite proyección anual lógica
aplicarSubidaAnual() actualiza el precio y recalcula impuestos
