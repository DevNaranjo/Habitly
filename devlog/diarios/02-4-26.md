# 📂 Diario de Desarrollo - Habitly v1.0.1
**Fecha:** 02-04-2026 | **Desarrollador:** DevNaranjo / Iriome

## 🚀 Hitos Alcanzados Hoy
¡La versión 1.0.1 ya es una realidad! Hemos blindado el motor principal antes del lanzamiento oficial.

### 1. 🛡️ Blindaje de Datos (Input Safety)
- **Corrección Crítica:** Se ha modificado el método `leerDouble` en `GestionInmobiliaria`.
- **Lógica:** Ahora el sistema rechaza valores `<= 0`, `,0` o entradas vacías, evitando errores de lógica en precios e IPC.

### 2. 🏦 Motor de Impuestos (Canarias-Only)
- **Polimorfismo Real:** Implementación obligatoria de `getPrecioFinalConImpuestos()` en `Piso` y `Casa`.
- **Cálculo:** Aplicación automática del **7% de IGIC** sobre el precio base.
- **Herencia:** Se ha cerrado el "contrato" de la clase abstracta `Vivienda`.

### 3. 🔄 Gestión de Estados Automática
- **Trigger de Pago:** El sistema ahora detecta cuando el `totalPagado` iguala o supera al precio con impuestos.
- **Auto-Update:** Cambio automático de estado de `DISPONIBLE` a `ALQUILADA` al completar el pago mensual.

### 4. 💾 Persistencia de Datos
- **Serialización:** Implementado el uso de archivos binarios `.dat` (`inventario_habitly.dat`).
- **Feedback:** El sistema ahora confirma visualmente cada vez que se guardan los datos con éxito.
