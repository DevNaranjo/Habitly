# 🏠 Habitly - Sistema de Gestión Inmobiliaria (v1.1.5 GUI)

**Habitly** es una aplicación profesional de escritorio diseñada para la gestión de activos inmobiliarios. Este repositorio contiene el instalador listo para usar en Windows.

---

## 🛠️ Instalación y Uso

La aplicación está lista para Windows y no requiere configurar entornos de desarrollo:

1. **Descarga el instalador**: Descarga el archivo `wizardSetUp.exe` de este repositorio.
2. **Ejecuta el asistente**: Haz doble clic en el instalador y completa los pasos en pantalla.
3. **Inicia Habitly**: Utiliza el acceso directo en el Escritorio o el menú Inicio para abrir la aplicación.
