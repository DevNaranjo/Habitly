# 🏠 Habitly - Property Management System (v1.1.5 GUI)

**Habitly** is a professional desktop application designed for comprehensive real estate asset management. This repository contains the ready-to-run Windows installer and metadata.

---

## 🛠️ Installation & Usage

The application is pre-packaged for Windows and does not require setting up a developer environment:

1. **Download the installer**: Download `wizardSetUp.exe` from this repository.
2. **Run the setup**: Double-click the installer and complete the steps on screen.
3. **Launch Habitly**: Use the Desktop shortcut or Start Menu icon to launch the application.
